# 非法捕捞声学监测仿真应用

> 一个基于声学感知的非法捕捞活动监测仿真程序的纯前端页面，使用 Vue 3 框架搭建。  
> 目前项目处于概念展示阶段，主要完成 UI 基础的搭建；后续将集成后端数据处理，使功能更加完善。

## 🚀 功能特点

- **通用声学监测**：支持自定义导入船舶引擎声和炸鱼声，实现通用化的非法捕捞声学监测。
- **仿真排布优化**：允许用户自定义船舶和麦克风位置，实时计算并提供 GDOP 与定位误差，帮助你探索最优麦克风布阵方案。
- **炸鱼事件检测**：可手动触发或由程序自动随机播放炸鱼音频，程序自主进行炸鱼事件检测。
- **事件时间线**：将关键检测事件按时间排布，清晰展示整个仿真过程中的重要节点。

## 🛠 技术栈

- **框架**：Vue 3
- **样式**：Tailwind CSS
- **构建工具**：Vite

## 📦 安装与运行

### 环境要求
- Node.js >= 18.0.0
- npm >= 9.0.0

### 项目依赖安装

```sh
npm install
```

#### 开发模式

```sh
npm run dev
```

#### 构建生产版本

```sh
npm run build
```

#### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
